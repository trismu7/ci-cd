About CI/CD Demo App

This is a demonstration project showcasing Continuous Integration and Continuous Deployment practices.

What is CI/CD?

CI/CD stands for Continuous Integration and Continuous Deployment:

  • Continuous Integration: Automatically building and testing code changes
  • Continuous Deployment: Automatically deploying code changes to production

Technologies Used

  • Python with Flask
  • GitHub Actions for CI/CD
  • Pytest for testing
  • HTML and CSS for the frontend
